# 🚀 Projectile Motion Simulator

A Python program that simulates two-dimensional projectile motion using the classical equations of motion. The simulator calculates key projectile parameters and visualizes the trajectory using Matplotlib.

## Features

- Calculates horizontal and vertical velocity components
- Computes maximum height, range, and time of flight
- Determines the time to maximum height
- Plots the projectile trajectory
- Supports multiple simulations in a single session

## Technologies

- Python
- NumPy
- Matplotlib
